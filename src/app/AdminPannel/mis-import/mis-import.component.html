Important

You upload multiple MIS files at a time.
ATS will proceed each MIS and add them to your database.

  • File format: MIS file should be standard .xls or .xlsx file
  • First Row Of MIS: First row of MIS file must be the default COLUMN NAME provided by source.
    You need to delete any blank or un-necessary rows existing above the column names for columns auto mapping.
  • Alert Popups: MIS file should not prompt any kind of popup alert.
  • Oldness: MIS file should not older than last 6 Month. It will prevent jobseekers records which are outdated. Your personal database should be latest.
  • MIS upload: You only need to upload your available recent MIS files and you never wait to complete the process. ATS keep clear record against this upload.
  • Auto Importing: After MIS submission, ATS will automatically start importing records from each file and also lookout for dupltcate profiles.
  • Proceeded Files: After MIS preceeded, ATS will delete MIS file from the server. ATS will never keep proceeded MIS file.
